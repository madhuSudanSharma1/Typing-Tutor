Download Keyboard

You can download keyboard from the menu, unzip it and follow the installation guide.

Select Keyboard

You can select Nepali Unicode Romanized Keyboard Layout from taskbar or Press Win + Space from your keyboard.

Mode Selection

You can select your preferred modes of game.

  • Practice Mode (Recommended)
  • Tutor Mode
  • Game Mode

User Control

  • Toggle Keyboard : To alter the visibility of keyboard layout.

  • Slider : To control the Zombie's speed

डाउन्लोड किबोर्ड

तपाईं मेनुबाट किबोर्ड डाउनलोड गर्न सक्नुहुन्छ, यसलाई अनजिप गर्नुहोस् र स्थापना गाइडलाई पछ्याउनुहोस्।

किबोर्ड चयन गर्नुहोस्

तपाइँ टास्कबारबाट नेपाली युनिकोड रोमनाइज्ड किबोर्ड लेआउट चयन गर्न सक्नुहुन्छ वा तपाइँको किबोर्डबाट Win + Space थिच्न सक्नुहुन्छ।

मोड चयन

तपाईं खेल को आफ्नो मनपर्ने मोड चयन गर्न सक्नुहुन्छ।

  • अभ्यास मोड (सिफारिस गरिएको)
  • ट्यूटर मोड
  • खेल्दै सिक्दै

प्रयोगकर्ता नियन्त्रण

  • किबोर्ड टगल गर्नुहोस् : किबोर्ड लेआउटको दृश्यता परिवर्तन गर्न।

  • स्लाइडर : जोम्बीको गति नियन्त्रण गर्न
